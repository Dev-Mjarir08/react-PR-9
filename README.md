## 📌 Project Overview

React PR-9 is a modern shopping cart application developed to demonstrate the practical implementation of **Redux Toolkit** with **React.js**.

The application fetches real-time product data from the **DummyJSON API**, stores it inside the Redux Store using **createAsyncThunk**, and displays products in an elegant card layout.

Users can dynamically load more products, reduce the visible list, and manage shopping cart items through centralized state management.

This project focuses on writing scalable React code while following modern best practices for component architecture, asynchronous API handling, and responsive UI design.

# 🚀 Application Workflow

```text
Application Starts
        │
        ▼
Redux Store Initialized
        │
        ▼
createAsyncThunk Calls API
        │
        ▼
Axios Fetches Products
        │
        ▼
Redux Store Updated
        │
        ▼
Products Displayed
        │
 ┌──────┴──────────┐
 ▼                 ▼
Load More      Add to Cart
 │                 │
 ▼                 ▼
Show Less     Cart Updated
                   │
                   ▼
          UI Automatically Re-renders
```

# 💻 Installation

Clone Repository

```bash
git clone https://github.com/Dev-Mjarir08/React-PR-9.git
```

Navigate to Project

```bash
cd React-PR-9
```

Install Dependencies

```bash
npm install
```

Run Development Server

```bash
npm run dev
```

Create Production Build

```bash
npm run build
```

Preview Production

```bash
npm run preview
```
